# 🧬 细胞生物学题库

基于《细胞生物学》教材整理的在线题库，包含**12章388道题**，覆盖选择题、判断题、填空题、名词解释、简答题、论述题六种题型。

## 在线预览

直接打开 `index.html` 即可使用（双击文件在浏览器中打开）。

## 部署到网站

### 方式一：GitHub Pages（免费，推荐）

1. 在 GitHub 上创建一个新仓库
2. 将本文件夹所有文件 push 到仓库中
3. 进入仓库 Settings → Pages → Source 选择 `main` 分支
4. 等待1-2分钟，网站会自动部署到 `https://你的用户名.github.io/仓库名/`

### 方式二：Netlify（免费）

1. 登录 [netlify.com](https://netlify.com)
2. 拖拽本文件夹到 Netlify 即可自动部署

### 方式三：Vercel（免费）

1. 登录 [vercel.com](https://vercel.com)
2. 导入本文件夹部署

## 功能特色

- ✅ **答题模式** — 选择题点击选项自动判断对错，判断题点击√/×按钮
- ✅ **按章节浏览**，按题型筛选
- ✅ **关键词搜索**
- ✅ **显示/隐藏答案**
- ✅ **随机抽题模式**（Ctrl+R 快捷键）
- ✅ **错题本** — 做错的题自动收录，可反复练习
- ✅ **标记掌握** — 已掌握的题可标记
- ✅ **学习统计** — 各章正确率、完成进度条、题型分布
- ✅ **夜间模式** 🌙 — 右上角一键切换
- ✅ 纯前端，零依赖，双击即可用
- ✅ 支持打印（打印时自动展开所有答案）
- ✅ 响应式设计，手机友好
- ✅ 学习数据自动保存（localStorage）

## 自定义数据

如需修改或增加题目，编辑 `data.js` 文件即可。数据格式：

```javascript
{
  chapter: "第一章 绪论",           // 章节名称
  questions: [
    {
      type: "选择题",               // 题型
      q: "题目内容",                // 题目
      opts: ["A、选项1", "B、选项2"], // 选项（仅选择题需要）
      answer: "A"                   // 答案
    }
  ]
}
```
